void setBarScale ([$barScale $barScale = 'a'], [$spaceBetweenBars $spaceBetweenBars = '4'], [$spaceBetweenGroups $spaceBetweenGroups = '8'])
  • $barScale $barScale: String You can specify widths and spacing absolutely or relatively, by entering one of the following values:
    • a: space_between_bars and space_between_groups are given in absolute units (or default absolute values, if not specified). Bars will be resized so that all bars will fit in the chart.
    • r: space_between_bars and space_between_groups are given in relative units (or default relative values, if not specified) Relative units are floating point values compared to the bar width, where the bar width is 1.0: for example, 0.5 is half the bar width, 2.0 is twice the bar width. Bars can be clipped if the chart isn't wide enough.
    Default value is 'a'
  • $spaceBetweenBars $spaceBetweenBars: Integer Space between bars in the same group. This is a width in pixels. Default value is 4 pixels for absolute values.
  • $spaceBetweenGroups $spaceBetweenGroups: Integer Space between bar groups in the same group. This is a width in pixels; Default value is 8 pixels for absolute values.
